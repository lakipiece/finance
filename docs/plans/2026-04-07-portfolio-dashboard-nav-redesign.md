# Portfolio & Ledger Improvements — Design Doc

Date: 2026-04-07

## 1. 대시보드 리디자인

**구조:**
- 상단: "전체" 카드 + 계좌별 카드 가로 나열 (선택 가능)
- 기본 선택: "전체"
- 카드 정보: 평가금액, 손익(원/%), 종목 수
- 카드 선택 시 하단에 종목별 보유현황 노출

**종목별 보유현황:**
- 티커: 뱃지(badge) 형태
- 종목명이 주 텍스트로 잘 보이게
- 금액: 전체 숫자 + 3자리 콤마, USD 소수점 2자리

## 2. 네비게이션 구조 변경

**변경 전:** `대시보드 | 스냅샷 | 수익 | 관리`
**변경 후:** `대시보드 | 스냅샷 | 수익 | 계좌관리 | 종목관리`

- `/portfolio/accounts` — 계좌관리 페이지 (신규)
- `/portfolio/securities` — 종목관리 페이지 (신규)
- 기존 `/portfolio/holdings`, `/portfolio/settings` → redirect

## 3. 계좌관리 개선

- 레이아웃 max-w-7xl
- 계좌 유형에 예금 / CMA 추가
- inputbox 글자 크기 text-xs
- 연결된 종목 저장 버그 수정
- 체크박스 리스트에서 연결된 종목이 상단 정렬

## 4. 종목관리 개선

- 레이아웃 max-w-7xl
- 기본 정렬: 국가 → 이름 순
- 카드 내 최근 수집 가격 표기
- 카드마다 가격 업데이트 버튼
- 상단 전체 가격 업데이트 버튼
- 가격 수집 이력 페이지 `/portfolio/securities/prices`
  - 종목별 날짜/가격 테이블
  - 종목 선택 시 가격 추이 라인 차트

## 5. 가계부

- 지출 내역 사용자 필드에 L / P 추가
