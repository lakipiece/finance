# Dashboard Redesign

## Date: 2026-04-04

## Goal
대시보드를 3행 구조로 단순화하고, 글로벌 필터 시스템을 통해 모든 차트/테이블이 연동되도록 개선.

## Layout

### 1행: KPI 카드
- `전체` | `고정비` | `변동비` | `여행공연비`
- 전체 카드: 연간 총지출 표시
- 분류 카드: 해당 분류 합계 + 전체 대비 비율(%)
- 클릭 시 분류 필터 토글 (전체 = 필터 해제)
- 월 필터 활성 시 KPI 수치도 해당 월 기준으로 변경

### 2행: 차트 + 드릴다운 (좌우 배치)
**좌측 — 차트:**
- 필터 해제(전체): 월별 스택바 차트 (고정비+변동비+여행공연비)
- 분류 선택: 해당 분류 월별 단일 바 차트
- 월 클릭 = 월 필터 토글

**우측 — 드릴다운 집계:**
- 필터 해제(전체): 카테고리별 비율 + 합계 (가로바)
- 분류 선택: 항목별 집계 (가로바 + 비율 + 금액)
- 항목 클릭 = 테이블 필터 (분류 > 항목)
- 상단에 내역 검색바

### 3행: 지출 내역 테이블
- 날짜, 분류, 내역, 비고, 결제수단, 금액 컬럼
- 날짜/분류/내역/금액 헤더 클릭 시 asc/desc 정렬
- 검색바, 페이지네이션 유지

## Filter System

| Filter   | Trigger             | Scope                                      |
|----------|---------------------|--------------------------------------------|
| 분류     | KPI 카드 클릭       | 전체 (KPI, 차트, 드릴다운, 테이블)         |
| 월       | 차트 막대 클릭      | 전체 (KPI 수치도 해당 월로 변경)           |
| 항목     | 드릴다운 항목 클릭  | 테이블만 필터                              |
| 검색     | 검색바 입력         | 테이블만 필터                              |
| 대출상환 | 관리탭 토글         | 전역 (모든 뷰에서 대출상환 카테고리 제외)  |

## Removed Components
- CategorySection (도넛차트 + 카테고리별 합계 바)
- CategoryDetailChart (전체 내역 TOP 10 바 차트)
- DrilldownPanel 상단 카테고리 카드 4개 (KPI 카드로 대체)

## Table Sort
- 정렬 가능 컬럼: 날짜, 분류, 내역, 금액
- 헤더 클릭 시 asc ↔ desc 토글
- 현재 정렬 상태 화살표 아이콘 표시
